# USER JOURNEY MAPS

## 1. Free Tier Student Journey
1. Visit landing page -> select Persian/English
2. Create account (email/phone) or trial mode
3. Choose stream and grade
4. Complete onboarding tutorial
5. Start first lesson (5E flow)
6. Receive immediate feedback + XP
7. Ask question in forum
8. Complete daily wellbeing check-in
9. Hit first milestone badge

## 2. Standard Tier Student Journey
Includes Free journey plus:
- joins group support session
- enters team challenge
- views detailed progress dashboard
- downloads selected offline content

## 3. Premium Tier Student Journey
Includes Standard journey plus:
- tutor matching and 1:1 session scheduling
- private session with whiteboard
- personalized learning plan updates
- human counselor access when needed

## 4. Parent/Guardian Journey
1. Create guardian account
2. Link child account
3. Set communication preferences
4. Review progress/wellbeing dashboard
5. Receive actionable recommendations
6. Book support/tutor where eligible

## 5. Support Teacher Journey
1. Access question bank and curriculum map
2. Build/modify lesson activities
3. Submit AI draft content for review
4. Monitor cohort analytics
5. Answer forum questions and moderate discussions

## 6. Private Tutor Journey
1. Complete profile and availability
2. Receive matched student requests
3. Conduct video sessions
4. Publish session notes and follow-up tasks
5. Track student improvement indicators

## 7. Platform Admin Journey
1. Monitor platform KPIs
2. Manage roles, plans, and compliance settings
3. Review moderation and safeguarding queue
4. Export institutional reports

## 8. Milestone Moments Across Journeys
- first completed unit
- first collaborative achievement
- grade-level completion
- certificate issuance and verification
