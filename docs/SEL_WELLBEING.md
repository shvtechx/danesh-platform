# SEL & WELLBEING FRAMEWORK

## 1. CASEL-Aligned Competencies
1. Self-awareness
2. Self-management
3. Social awareness
4. Relationship skills
5. Responsible decision-making

Each unit maps at least one SEL competency and one explicit learner behavior.

## 2. Wellbeing Check-in Design
- Daily quick check-in for students
- Age-banded UX:
  - KG–2: emoji faces + optional voice note
  - 3–6: mood + energy + short prompt
  - 7–12: mood, stress, belonging, confidence scales
- Automated escalation thresholds for risk indicators

## 3. Trauma-Informed Principles
- Predictable routines and safe micro-interactions
- Non-triggering language and visual defaults
- User control: pause/skip, private reflection spaces
- Clear support pathways and human handoff

## 4. Peer Mentoring (جفت‌یادگیری)
- Opt-in buddy matching by grade/language
- Structured weekly check-ins
- Safeguarded communications and moderation rules

## 5. Digital Citizenship Integration
- Respectful communication, privacy awareness, misinformation literacy
- Embedded scenarios and reflection tasks

## 6. Anti-Bullying Protocol
1. Detect (AI + user report + moderator review)
2. Triage severity
3. Respond (content action + support)
4. Document and follow-up
5. Restore community climate

## 7. Counselor Support Model
- Tier 1/2: counselor bot + escalation queue
- Tier 3: scheduled human counselor sessions
- Safeguarding and confidentiality controls

## 8. Teacher/Staff Wellbeing
- Resource hub for burnout prevention and referral pathways
- Professional learning on trauma-sensitive facilitation
