# GAMIFICATION DESIGN

## 1. System Goals
- Increase persistence and mastery, not shallow competition
- Reinforce collaboration, wellbeing, and growth mindset
- Provide culturally relevant narratives

## 2. XP Model

### Base XP Formula
`XP = ActivityBase × DifficultyMultiplier × QualityMultiplier + StreakBonus + CollaborationBonus`

| Activity | Base XP |
|---|---:|
| Micro-lesson completion | 10 |
| Practice set completion | 20 |
| Quiz pass (>=70%) | 30 |
| Project submission | 60 |
| Peer-help accepted answer | 15 |
| Daily check-in streak | 5 |

Difficulty multiplier: Easy 1.0, Medium 1.2, Hard 1.5

## 3. Level Progression
- Non-linear progression to keep early motivation high
- Example curve: `RequiredXP(level n) = 100 * n^1.35`

## 4. Badge Framework (نشان‌ها)
- **Mastery badges:** topic mastery thresholds
- **Helper badges:** forum/study-group contribution quality
- **Creativity badges:** project originality rubric
- **Perseverance badges:** retries + growth over time

## 5. Narrative Quests (ماموریت‌های داستانی)
Quest arcs:
1. Persian mythology-inspired journeys
2. Islamic Golden Age scholar missions (e.g., Khwarizmi, Ibn Sina)
3. Global science/citizenship missions

Each arc contains: intro story, curricular tasks, collaborative checkpoint, reflection, milestone reward.

## 6. Team Challenge Mechanics
- Team sizes: 3–6 learners
- Mixed skill composition encouraged
- Team score combines mastery gain + collaboration quality
- Anti-free-rider rules using participation logs

## 7. Leaderboards
- Optional by user setting
- Default views emphasize **personal best growth** and **class contribution**, not raw rank
- Age-appropriate visibility controls

## 8. Virtual Economy
- Earn coins via verified learning behaviors
- Spend on avatars, profile themes, virtual room items
- Guardrails: no pay-to-win, no trading with real money

## 9. Milestone & Certificate Triggers
- Unit mastery milestone
- Grade-band completion milestone
- Community helper milestone
- Portfolio excellence milestone

Certificates include unique verification code + URL.

## 10. Abuse Prevention
- Bot detection on repetitive events
- XP cooldowns for exploitable loops
- Moderator override and fraud audit log
